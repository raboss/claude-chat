نظرة عامة على النظام

ALUMINOR هو نظام تطبيق ويب تقدمي (PWA) متكامل مصمم خصيصاً لإدارة مشاريع الألمنيوم والزجاج. يغطي النظام دورة حياة المشروع كاملة من الإنشاء وحتى التسليم، مع إدارة القياسات والتركيبات والتقارير والمستندات.

6 صفحات رئيسية
60+ صلاحية دقيقة
6 قوالب مستخدمين
20 ثانية مزامنة

🏭 نوع النظام

تطبيق ويب تقدمي (PWA) يعمل على جميع الأجهزة بما فيها الهواتف والأجهزة اللوحية وأجهزة الكمبيوتر

💻 التقنية المستخدمة

Node.js + Express للسيرفر، HTML/CSS/JS خالص للواجهة، تخزين بيانات عبر ملفات JSON

🌐 التوافقية

يعمل على جميع المتصفحات الحديثة، قابل للتثبيت كتطبيق، يدعم العمل بدون إنترنت

🔐 الأمان

نظام صلاحيات على مستوى الأزرار، علامات مائية، سجل نشاطات، تحكم بالوصول

تفاصيل الوحدات الحالية

📋

المشاريع - Projects

لوحة التحكم الرئيسية التي تعرض جميع المشاريع على شكل بطاقات مع إحصائيات شاملة وأدوات بحث وتصفية متقدمة.

  • بطاقات مشاريع تفاعلية
  • إحصائيات (إجمالي/نشط/تركيب/تسليم)
  • شريط أدوات (مشروع جديد، تقارير)
  • استيراد وتصدير Excel
  • لوحة جانبية بـ 13 زر إجراء
  • بحث وفلاتر متقدمة
  • فرز حسب التاريخ والحالة
  • عرض تفاصيل المشروع
📊

التقارير - Reports

نظام تقارير شامل يتضمن 5 تبويبات لتغطية جميع جوانب المشاريع بالرسوم البيانية والجداول الزمنية.

  • نظرة عامة مع رسوم بيانية تفاعلية
  • تقارير أسبوعية تفصيلية
  • تقارير شهرية شاملة
  • تقارير العملاء
  • خط زمني / مخطط جانت
🗃

السجلات - Records

أرشيف شامل للتقارير المحفوظة مع إمكانية العرض والطباعة والحذف.

  • تقارير الإنتاج المحفوظة
  • التقارير الأسبوعية المحفوظة
  • تقارير المواد المحفوظة
  • عرض وطباعة وحذف التقارير
📄

المستندات - Documents

إنشاء المستندات الرسمية مع شعارات الشركة والعلامات المائية.

  • مستخلصات المشاريع
  • أوامر العمل
  • علامات مائية تلقائية
  • شعارات الشركة
  • طباعة واستخراج PDF
  • قوالب مستندات متعددة
🔧

التركيبات - Installations

وحدة شاملة لإدارة عمليات التركيب بأربعة تبويبات رئيسية.

📏 مقاسات (Measurements)

  • إضافة قياسات العميل من الموقع
  • إرفاق صور من الموقع
  • ربط القياسات بالمشروع

🚫 نواقص (Defects)

  • تتبع الزجاج المكسور أو المفقود
  • تتبع الألمنيوم التالف
  • تتبع الإكسسوارات الناقصة

🚚 أوامر تسليم (Delivery Orders)

  • إنشاء أوامر التسليم
  • تأكيد بتوقيع العميل
  • تحديث حالة المشروع تلقائياً

🛠 صيانة (Maintenance)

  • إنشاء أوامر الصيانة
  • توزيع على الفنيين
  • شهادة تسليم تلقائية عند الإتمام

نظام الصلاحيات

يتميز النظام بنظام صلاحيات دقيق للغاية يتحكم في أكثر من 60 إذن على مستوى الأزرار الفردية. كل مستخدم يمكن تخصيص صلاحياته بشكل مستقل أو استخدام أحد القوالب الجاهزة.

القالب الصلاحيات الوصف
مدير النظام جميع الصلاحيات وصول كامل لجميع الوحدات والإعدادات بدون قيود
مشرف إدارة + تقارير إدارة المشاريع والفرق مع صلاحيات الاعتماد والتقارير
فني تركيبات + قياسات إدخال القياسات وتأكيد التسليمات والصيانة
مبيعات مشاريع + عملاء إنشاء المشاريع وإدارة العملاء وعروض الأسعار
محاسب مستندات + تقارير مالية إدارة المستخلصات والتقارير المالية
مشاهد فقط عرض فقط عرض البيانات بدون إمكانية التعديل أو الحذف

🎯 فلترة حسب الشركة/المنطقة/المعرض

كل مستخدم يرى فقط المشاريع المرتبطة بالشركة والمنطقة والمعرض المخصص له

💧 تحكم بالعلامة المائية

إمكانية تفعيل أو تعطيل العلامات المائية لكل مستخدم على حدة

واجهات البرمجة (API)

يوفر النظام مجموعة شاملة من واجهات البرمجة RESTful لجميع العمليات:

GET /api/projects جلب جميع المشاريع
POST /api/projects إنشاء مشروع جديد
PUT /api/projects/:id تحديث مشروع
DEL /api/projects/:id حذف مشروع
GET /api/settings جلب الإعدادات
POST /api/settings حفظ الإعدادات
GET /api/users جلب المستخدمين
POST /api/login تسجيل الدخول
GET /api/reports جلب التقارير
POST /api/reports حفظ تقرير
GET /api/installations جلب بيانات التركيبات
POST /api/installations حفظ بيانات التركيبات
POST /api/backup إنشاء نسخة احتياطية
POST /api/restore استعادة نسخة احتياطية
GET /api/activity-log سجل النشاطات

💾
نسخ احتياطي تلقائي

خارطة التطوير المستقبلية

خطة تطوير مرحلية شاملة لتوسيع قدرات النظام وإضافة وحدات جديدة تغطي جميع جوانب إدارة مشاريع الألمنيوم والزجاج.

المرحلة الأولى

📐 الرسومات والمخططات التنفيذية

  • صفحة خاصة بالرسومات الهندسية
  • رفع مخططات AutoCAD/PDF
  • ربط الرسومات بالمشاريع
  • معاينة المخططات داخل النظام
  • تتبع نسخ المخططات (Versioning)
  • اعتماد المخططات من المشرف
المرحلة الثانية

🏭 أوامر التصنيع

  • صفحة أوامر تصنيع الألمنيوم
  • صفحة أوامر تصنيع الزجاج
  • ربط أوامر التصنيع بالمشاريع والمقاسات
  • تتبع حالة التصنيع (معلق/جاري/مكتمل)
  • تقارير إنتاج المصنع
  • جدول زمني للتصنيع
المرحلة الثالثة

💰 العهدة والرواتب

  • صفحة إدارة العهد (أدوات، معدات، مواد)
  • تسليم واستلام العهد للفنيين
  • تتبع العهد المفقودة أو التالفة
  • صفحة الرواتب والمستحقات
  • كشوف الرواتب الشهرية
  • الحوافز والخصومات
المرحلة الرابعة

👥 فرق التركيبات

  • صفحة إدارة الفرق
  • تعريف الفرق (اسم الفرقة، رئيس الفرقة، الأعضاء)
  • توزيع المشاريع على الفرق
  • جدول عمل الفرق
  • تقارير أداء الفرق
  • تتبع مواقع الفرق
المرحلة الخامسة

🏢 الفروع

  • صفحة إدارة الفروع
  • كل فرع له مشاريعه وفرقه
  • تقارير منفصلة لكل فرع
  • مقارنة أداء الفروع
