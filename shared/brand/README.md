# NouRion — Brand Identity Guide

## 🔬 مفهوم العلامة

**NouRion** = **Nour** (النور/الضوء) + **Ion** (الأيون/الذرة)

اسم يجمع بين الإضاءة والطاقة الذرية، مع حرف **R** مميز أزرق يربط الكلمتين. رمز الذرة في النهاية يعبر عن الدقة العلمية والذكاء الاصطناعي.

### القيم البصرية
- **الدقة**: التصميم العلمي للذرة
- **الحركة**: دوران الإلكترونات يرمز للتطور المستمر
- **الذكاء**: النواة المركزية = قلب المعالجة الذكية
- **الشمولية**: 3 مسارات = 3 تخصصات (ألمنيوم، حديد، خشب، UPVC)

| 🩵 **Sky Bright** | `#3FA2E0` | اللون الثانوي — المسار الأمامي، الأزرار |
| 🟠 **Amber Electron** | `#F4A91E` | نقاط الطاقة — الإلكترونات، التنبيهات |
| 🔴 **Coral Nucleus** | `#E63946` | التحذيرات، الحذف، النواة |
| ⚫ **Cosmos** | `#0A1628` | النصوص، الخلفية الداكنة |

### قواعد الاستخدام

1. **Navy + Sky** = الأزرار الأساسية والروابط
2. **Amber** = للتنبيهات الإيجابية والعناصر التفاعلية المميزة
3. **Coral** = فقط للحذف والتحذيرات الحرجة
4. **60-30-10 Rule**:
   - 60% خلفيات محايدة (أبيض/رمادي فاتح)
   - 30% أزرق (Navy + Sky)
   - 10% أكسنتات (Amber + Coral)

## 🔤 اللوجو (Wordmark)

### الهيكل
```
Nou R ion  [⚛]
 │   │  │    │
 │   │  │    └── رمز الذرة (24px)
 │   │  └────── "ion" — navy deep 700
 │   └───────── "R" — gradient blue, weight 800, +8px
 └───────────── "Nou" — navy deep 700
```

### المقاييس
- الحد الأدنى: 120px عرض (للويب)
- Clear space: ≥ ارتفاع حرف R من كل الجهات
- لا تستخدم اللوجو على خلفيات ملونة بدون خلفية بيضاء دائرية

### الاستخدامات الممنوعة
- ❌ تدوير اللوجو
- ❌ تغيير اللون الأزرق للـ R
- ❌ إضافة ظلال غير موجودة
- ❌ فصل الذرة عن النص
- ❌ استخدام فونت مختلف

## 🌀 Loading Animation Principles

### المبدأ الأساسي
حركة الإلكترونات على المسارات البيضاوية ترمز لـ:
- **المعالجة الجارية** (Processing)
- **الدقة العلمية** (Precision)
- **الذكاء الحاسوبي** (Intelligence)

### التوقيت
- نواة مركزية: نبض 2 ثانية (breathing pulse)
- إلكترون 1: دوران 3.2s
- إلكترون 2: دوران 4.0s
- إلكترون 3: دوران 2.6s

> الاختلاف في السرعة يخلق حركة "غير منتظمة"، وهو ما يميز الذرات الحقيقية (لا تدور الإلكترونات في أزمنة متطابقة).

## 📐 مبادئ التصميم

### 1. Clarity First (الوضوح أولاً)
- مساحات بيضاء سخية
- تسلسل هرمي واضح
- معلومة واحدة لكل سطر بصري

### 2. Scientific Precision (الدقة العلمية)
- شبكة 8pt
- محاذاة مثالية
- لا عناصر عائمة بدون سبب

### 3. Tech but Warm (تقني لكن دافئ)
- البرتقالي يكسر برودة الأزرق
- زوايا ناعمة (radius 8-12px)
- ظلال خفيفة

### 4. Accessibility (إمكانية الوصول)
- تباين AA+ لكل النصوص
- حالات تركيز واضحة
- دعم كامل للوحة المفاتيح
- RTL/LTR كامل

## 🌍 متعدد اللغات

### الاتجاهات
- العربية: RTL
- الإنجليزية: LTR
- الهندية: LTR

### الفونتات المختلطة
```css
body {
  font-family: var(--nr-font-ar), var(--nr-font-en);
}
:lang(en) { font-family: var(--nr-font-en); }
:lang(hi) { font-family: 'Noto Sans Devanagari', sans-serif; }
```

### Font Size Adjustments
- العربية أكبر بـ 8% (حروف أقصر)
- الهندية أكبر بـ 5%
