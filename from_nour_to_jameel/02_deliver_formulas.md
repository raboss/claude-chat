## الخبر الحلو

## المطلوب منك

محمود بده نحوّل يلي تعلّمته لكود يشتغل بالبرنامج. البرنامج عنده:
- ✅ استمارة تصنيع
- ✅ طلبات شراء
- ✅ مستخلص
- ✅ مقاسات + مساحات (أوتوماتيك)

**الناقص يلي بدنا إياه منك:**

### 1. أنواع القطاعات + عددها
- حسب مدخلات المقاسات + نوع القطاع + المخطط المعتمد
- اسم القطاع + العدد
- مثال: SARAYA 10 / 2 PANEL SLIDING → كم قطعة frame, panel, fly, etc.

### 2. أوامر التقطيع (Cutting List)
- لكل نوع قطاع: أطوال القص بالمللي
- الصيغ يلي استخرجتها (Frame W = W + 42, Panel W = (W+42)/2 - 48, إلخ)
- **لكل نوع قطاع درسته** — مش بس Saraya

### 3. أوامر الزجاج (Glass Orders)
- مقاسات الزجاج لكل فتحة
- Glass W, Glass H لكل نوع قطاع

## الشكل يلي بدي إياه

سلّمني الداتا بـ **JSON** بهالشكل (أو قريب منو):

```json
{
  "sections": {
    "S1_SARAYA_10_2PANEL": {
      "name_ar": "سرايا 10 - سحاب لوحتين",
      "name_en": "Saraya 10 - 2 Panel Sliding",
      "type": "sliding",
      "formulas": {
        "frame_w": "W + 42",
        "frame_h": "H + 42",
        "panel_w": "(W + 42) / 2 - 48",
        "panel_h": "H - 75",
        "fly_w": "(W + 42) / 2 - 21",
        "fly_h": "H - 68",
        "glass_w": "(W + 42) / 2 - 119",
        "glass_h": "H - 175"
      },
      "parts": [
        { "name": "حلق علوي", "formula": "frame_w", "qty_per_unit": 1 },
        { "name": "حلق سفلي", "formula": "frame_w", "qty_per_unit": 1 },
        { "name": "حلق جانبي", "formula": "frame_h", "qty_per_unit": 2 }
      ],
      "area_formula": "frame_w * frame_h / 1000000"
    }
  }
}
```

### المهم:
- **كل قطاع درسته** — حطو بالـ JSON
- **الـ 42 قطاع** يلي عندك — حتى لو ما درست صيغهم كلهم، حط يلي عندك
- **الأجزاء (parts)** لكل قطاع — شو القطع يلي بتطلع منو
- **صيغ الزجاج** لكل قطاع

حط الملف بـ `shared/sections_formulas.json`
